# **App Name**: Diamond Digger

## Core Features:

- Task List: Display a list of tasks (e.g., rate app, watch ads, use app for a duration) with corresponding diamond rewards.
- Balance Tracking: Track user's diamond and INR (₹) balance, updating in real-time as tasks are completed.
- Rewarded Ads: Integrate rewarded video ads that award diamonds upon completion.
- Interstitial Ads: Implement interstitial ads that appear when the 'Earn' button is pressed, balancing monetization with user experience.
- Cashout: Enable the 'Cashout' button when the user reaches the minimum payout threshold (₹100).
- Progress Bar: Visually represent the user's progress towards the minimum cashout amount with a progress bar.
- Ad Mediation Tool: Use generative AI as a tool that reasons and decides whether to show an Ad depending on various factors such as ad frequency and the amount of coins the user currently possesses.

## Style Guidelines:

- Primary color: Vibrant green (#4CAF50) to evoke a sense of growth and earning.
- Background color: Light grey (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Golden yellow (#FFC107) to highlight rewards and cashout options.
- Body and headline font: 'PT Sans', a humanist sans-serif font for a modern and approachable look. Using a font size of 16px for body text and 24px for headlines will enhance clarity.
- Use diamond-themed icons for rewards and currency, maintaining a consistent and clean design.
- Employ a card-based layout for tasks to enhance clarity and organization.
- Incorporate subtle animations when tasks are completed and balances are updated.
